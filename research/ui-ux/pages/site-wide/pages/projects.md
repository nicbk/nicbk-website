# Projects Page

Status: Decided 2026-07-02.

Simple list of sub-applications. Each entry is a single link containing:

- **Name** (primary text color/weight).
- **One-line description** immediately after, in a slightly darker/dimmer
  text color than the name — the same emphasis-by-color-weight convention
  used for secondary text in
  [blog-page.png](../../../../../high-level-guidance/design/blog-page.png)
  (e.g. its post dates), applied here so the description reads clearly as
  secondary to the name without needing a separate visual element.

The name + description together form one clickable link to that
sub-application. No card/grid layout, no extra metadata (tech stack, status
tags, etc.) — keep it to name + description per the design philosophy's
low-friction/avoid-overcomplicating guidance.

Content at launch: a single entry, linking to the lit tracker. Uses the
[site header](../components/header.md).
